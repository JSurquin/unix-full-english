# Module 1 answers 📝

**Answer 1:** B) Ken Thompson and Dennis Ritchie
- Unix was created in 1969 at Bell Labs (AT&T)

**Answer 2:** B) GNU's Not Unix
- A recursive acronym coined by Richard Stallman

**Answer 3:** B) The core of the system that manages hardware
- The kernel is the interface between hardware and software

**Answer 4:** B) Devices and processes are accessed like files
- Unix uses the file abstraction for devices (`/dev`), processes (`/proc`), etc.

**Answer 5:** C) Debian
- Debian is known for stability, especially on servers

---

# Module 1 answers (continued) 📝

**Answer 6:** B) The interface between the user and the kernel
- The shell (bash, zsh, etc.) interprets commands

**Answer 7:** B) Build simple programs that do one thing well
- Unix philosophy: modular, composable programs with pipes

**Answer 8:** C) AT&T Bell Labs
- Unix was developed at Bell Labs in 1969

**Answer 9:** A) A complete operating system based on the Linux kernel
- A distribution includes the kernel + GNU + tools + package manager

**Answer 10:** C) It is open source and free
- Open source enables transparency, security, and freedom
