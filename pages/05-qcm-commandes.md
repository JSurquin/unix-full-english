# Module 2 answers üìù

**Answer 1:** B) `pwd`
- pwd = Print Working Directory

**Answer 2:** B) List with details and hidden files
- `-l` = long format, `-a` = all (include hidden)

**Answer 3:** A) `mkdir -p /full/path`
- `-p` = parents, creates intermediate directories

**Answer 4:** B) `cp -r source dest`
- `-r` = recursive, for copying directories

**Answer 5:** C) Prints the file contents
- cat = concatenate, prints content

# Module 2 answers (continued) üìù

**Answer 6:** B) `tail file`
- tail shows the end, head shows the beginning

**Answer 7:** B) Output redirection (overwrite)
- `>` redirects and overwrites, `>>` appends

**Answer 8:** C) `grep error *.log`
- grep searches for patterns in files

**Answer 9:** A) Finds all `.txt` files from the root
- find searches files by criteria

**Answer 10:** B) `cmd1 | cmd2`
- The pipe `|` sends cmd1‚Äôs stdout to cmd2‚Äôs stdin
